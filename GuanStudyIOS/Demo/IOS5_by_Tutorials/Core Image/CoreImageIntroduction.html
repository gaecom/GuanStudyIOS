Core Image 是一个强大的框架,它可以让你轻松地使用图像滤镜,例如修改饱和度,色 调或者曝光。它能迅速的使用 GPU(或者 CPU,用户定义)处理图像数据。快到可以实时 地处理视频帧! Core Image 滤镜能够一次给予一张图像或者视频帧多种视觉效果。当多个滤镜叠加起来 也是有效的。因为它们创建了一个可修改的单一滤镜应用到这张图像上,而不是通过各个滤 镜一次一个地处理该图像。 每个滤镜都有它自己的参数,可以在代码中查询该滤镜提供的信息,比如它的作用和输 入参数。系统也可以检查出哪些滤镜是必要的。此时,仅有部分在 Mac 上可用的 Core Image 滤镜在 iOS 设备上是有效的。然而,越来越多的有效的 API 可以被用作去发现滤镜新 的特性。

  1. CIContext: 一张图像上的所有处理都是在一个 CIContext 里实现。这有些类似于一个 Core Graphics 或者 OpenGLcontext。

  2. CIImage: 该类保存了图像数据。可以通过 UIImage,图像文件或像素数据来创建。

  3. CIFilter: 该滤镜类用一个字典定义了特定滤镜的详细属性。这些滤镜有 vibrancefilters,color inversion filters,cropping filters 等等。 我们将会在我们创建的项目中使用这些类。
